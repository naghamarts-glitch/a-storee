# PHP Backend API Implementation for nagham777 DB

## Steps:
1. [ ] Create config/db.php - PDO connection
2. [ ] Create api/users/register.php - POST register
3. [ ] Create api/users/login.php - POST login  
4. [ ] Create api/books/get.php - GET all books
5. [ ] Create api/books/add.php - POST add book
6. [ ] Create api/books/delete.php - POST/DELETE book by id
7. [ ] Create api/products/get.php - GET all products
8. [ ] Create api/products/add.php - POST add product
9. [ ] Create api/products/delete.php - POST/DELETE product by id
10. [x] Test: Start XAMPP, use Postman/curl on localhost/a-store-main/api/...
